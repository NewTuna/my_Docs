## Samba Install (Ubuntu 14.04 기준)
   
1. 삼바 설치
  * sudo apt-get install samba

2. 삼바 사용자 등록
  * sudo smbpasswd -a <사용자명>
  * New SMB password 입력 및 재입력

3. 삼바설정파일 수정
  * sudo vim /etc/samba/smb.conf
  
  > [samba] <br/>
  > comment = samba <br/>
  > path = /home/samba/my_repo <br/>
  > valid users = d1, d2 <br/>
  > pulbic = yes <br/>
  > writable = no <br/>
  > read only = yes <br/>
  > geust ok = yes <br/>
